Musical Complexity Scoring

Musical Complexity Scoring is a project designed to give anyone related to music performance an objective measurement of the complexity -- or difficulty -- of any piece of music. To our knowledge, it is the first such objective measurement of its kind. It works by assigning difficulty values to specific musical elements: notes and intervals receive point values between 1 and 10; articulations, dynamics, tempos, and key signatures receive multiplier values greater than 0.

By scanning the MusicXML representation of a piece and referencing the preset difficulty values, the backend code can form an aggregate complexity score. This web page then displays that overall score along with some other relevant data, such the toughest measure of music in the piece (the one with the highest individual score) and the breakdown of notes and their multipliers versus intervals and their multipliers.

This algorithm works for single-part pieces as well as multi-part pieces, such as for conductors or large ensembles. However, the current implementation only specifies one set of difficulty values (for beginner clarinet) so all pieces are analyzed against that set, regardless of instrument. As we gather more feedback from experts, further difficulty values will be specified and existing values will be honed to precision.

For more information about this work, please check out the research abstract we submitted to Interdisciplinary Research Day 2015 at VT or go directly to the source code on github from the links on the navigation bar.
